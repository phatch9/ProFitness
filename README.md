[Building and Running]
Use Python 3.8-3.10
To run the flask application, run these commands below
```
git clone https://github.com/JimSongTheGreatest/FitPal.git

cd FitPal

pip3 install -r requirements.txt

python3 run.py

```
after that, open the url:
http://localhost:5001/
